# Licitações / Registros de Aquisição — CoreTech

Este arquivo contém um template para registrar processos de aquisição, licitações ou recursos financeiros relacionados ao projeto (ex.: compra de hardware para guarita, contratação de hospedagem, serviços terceirizados).

Modelo sugerido
- Título do processo:
- Tipo: (Compra direta / Licitação / Contratação de serviço)
- Data de início:
- Órgão/Responsável:
- Valor estimado / valor contratado:
- Descrição resumida:
- Documentos anexos (links para PDFs, ofícios, contratos):
- Status: (Proposta / Em andamento / Concluído)

Onde colocar
- Guarde documentos oficiais em `/docs/procurement/` (um subdiretório) e um resumo em `/docs/PROCUREMENT.md`.

Exemplo de uso
- Arquivo `/docs/procurement/licitacao_hospedagem_2026.pdf`
- Entrada resumida em `PROCUREMENT.md` com link para o PDF.
